# Contributing to LRDEnE Specialist Agents Kit

Thank you for your interest in contributing to the Specialist Agents Kit! Our goal is to provide the most professional, high-performance workstation for AI-assisted development.

## How to Contribute

1.  **Report Bugs**: If you find an issue, please open a GitHub Issue with clear reproduction steps.
2.  **Suggest Features**: We love new specialist ideas! Open an issue to discuss.
3.  **Code Changes**:
    -   Fork the repository.
    -   Create a feature branch (`feat/your-feature`).
    -   Ensure all MCP tools are type-safe and use Zod for validation.
    -   Submit a Pull Request with a detailed summary of changes.

## Development Setup

1.  Clone the repo.
2.  Run `bash setup.sh` to initialize the environment.
3.  Work in the `mcp-server/src` directory.
4.  Run `npm run dev` in `mcp-server/` to watch for changes.

## Standards

-   **Clean Code**: No over-engineering. Self-documenting.
-   **Security**: Never commit secrets or API keys.
-   **Documentation**: All new tools must have JSDoc and appear in the `README.md`.
