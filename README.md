# LRDEnE Specialist Agents Kit

A premium, portable pack of AI agents designed to transform your IDE into a high-performance specialist workstation. Compatible with **Cursor**, **Windsurf**, **Antigravity**, **Lovable**, and **Bolt**.

## Included Specialists

1.  **WebApp Architect**: End-to-end web application development, from scaffolding to deployment.
2.  **SEO Growth Auditor**: Technical and on-page SEO audits, schema generation, and rank optimization.
3.  **GDPR Compliance Copilot**: Privacy-by-design reviews, data mapping, and compliance gap reporting.

## Core Features

*   **Shared MCP Server**: A robust TypeScript-based Model Context Protocol server providing 10+ professional tools.
*   **Host-Agnostic Core**: Unified YAML definitions ensure consistent behavior across all platforms.
*   **Platform Adapters**: Tailored rules, skills, and workflows for every major AI-integrated development environment.
