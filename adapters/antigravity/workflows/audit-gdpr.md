# Antigravity Workflow: Audit GDPR

1. Activate the `GDPR Compliance Copilot` skill.
2. Collect from user: product description, jurisdictions, vendor list.
3. Run `data_flow_map` to trace all personal data touchpoints.
4. Run `gdpr_gap_report` to surface compliance gaps.
5. Run `privacy_policy_inputs` to structure draft policy data.
6. Run `dsar_pack` to generate DSAR handling templates.
7. Compile full `gdpr-gap-report.md`.
8. Include legal disclaimer from `core/policies/legal-disclaimer.md` in ALL outputs.
