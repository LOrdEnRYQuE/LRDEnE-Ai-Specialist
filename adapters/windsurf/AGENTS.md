# Specialist Agents

This repository is powered by the Specialist Agent Pack.

## Active Agents

- **WebApp Architect**: For end-to-end development.
- **SEO Growth Auditor**: For SEO audits and optimization.
- **GDPR Compliance Copilot**: For privacy and compliance reviews.

## Custom Workflows

- `/build-feature`: Trigger WebApp Architect flow.
- `/audit-seo`: Trigger SEO Auditor flow.
- `/audit-gdpr`: Trigger GDPR Copilot flow.
