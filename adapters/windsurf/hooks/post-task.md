# Windsurf Hooks — Post-Task

After every specialist task completes, verify:

1. **Tests**: Were relevant tests run? If not, flag as incomplete.
2. **Docs**: Has any documentation impacted by the change been updated?
3. **Summary**: Produce a short summary of: what changed, files touched, how to verify manually, and known limitations.
4. **PR Description**: Prepare a pull request description ready for review.
