# Windsurf Workflow: Audit GDPR

1. Define product scope: jurisdictions, data categories, user types.
2. Run `data_flow_map` to trace personal data touchpoints.
3. Run `gdpr_gap_report` to identify compliance gaps.
4. Run `privacy_policy_inputs` to collect draft inputs.
5. Run `dsar_pack` for DSAR handling templates.
6. Compile all findings into `gdpr-gap-report.md`.
7. Always include legal disclaimer — human review is required before using any output.
