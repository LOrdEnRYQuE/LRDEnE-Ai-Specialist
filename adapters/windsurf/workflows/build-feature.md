# Windsurf Workflow: Build Feature

1. Run `repo_scan` to understand the project structure.
2. Produce `implementation-plan.md` and confirm with the user.
3. Create branch `feature/<name>`.
4. Scaffold files using `scaffold_feature`.
5. Implement logic in small, focused commits.
6. Run tests. Fix any failures before proceeding.
7. Update documentation.
8. Produce PR description using the post-task hook.
