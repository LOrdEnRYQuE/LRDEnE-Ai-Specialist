# Workflow: Audit SEO (SEO Growth Auditor)

1.  **Crawl Site**: Use `crawl_site` to map out the page structure and extract metadata.
2.  **Run Audit**: Use `seo_audit` to identify technical issues (404s, missing titles, layout shifts).
3.  **Analyze Intent**: Map each crawled page to a primary search intent.
4.  **Identify Gaps**: Use `content_brief_generator` to find keyword or content opportunities.
5.  **Propose Markup**: Use `schema_markup` to generate relevant JSON-LD records.
6.  **Internal Linking**: Use `internal_link_map` to find optimization opportunities.
7.  **Summary Report**: Produce a comprehensive `seo-audit.md` with prioritized action items.
