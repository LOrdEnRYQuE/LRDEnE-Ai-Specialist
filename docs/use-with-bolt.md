# Using Specialist Agents with Bolt

1.  **Agent Config**: Bolt allows for custom agent definitions. Use the YAML files in `core/agents/` as the blueprint.
2.  **MCP Tools**:
    - Register the **Specialist Agents MCP Server** in your Bolt environment.
3.  **Execution**: When building new features, prompt Bolt to "Act as the WebApp Architect specialist."
