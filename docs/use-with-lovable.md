# Using Specialist Agents with Lovable

1.  **GitHub Connection**: Connect your project repository to Lovable.
2.  **MCP Integration**:
    - Deployment of the MCP server to a public URL is recommended for Lovable integration.
    - Add the MCP endpoint to your Lovable project configuration.
3.  **Prompting**: Use the instructions in `adapters/lovable/prompt-templates/` to guide the Lovable agent.
